Experiment task is assigned in [Task](https://github.com/hustwyk/Compiler-theory-experiment/blob/master/2016%E5%B9%B4%E8%AE%A1%E7%AE%97%E6%9C%BA%E7%A7%91%E5%AD%A6%E4%B8%93%E4%B8%9A%E7%BC%96%E8%AF%91%E5%8E%9F%E7%90%86%E5%AE%9E%E9%AA%8C%E4%BB%BB%E5%8A%A1%E4%B9%A6.doc).

This experiment is consisted of two parts.

In _test1_,U can double click the `a.exe` to see the result or use commentline to compile `a.l` and U can write the result in `a.txt`(U can refer the _.bat_ document in _test2_ [bat.bat](https://github.com/hustwyk/Compiler-theory-experiment/blob/master/test2/bat.bat)).
